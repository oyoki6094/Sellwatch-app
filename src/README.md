# Structure du dossier src – Sellwatch

Ce dossier contient tout le code source de l'application Sellwatch.

## Organisation

- **components/** : composants UI réutilisables (boutons, cartes, etc.)
- **pages/** : pages principales de l'application (Dashboard, Login, etc.)
- **services/** : logique métier et appels API
- **hooks/** : hooks React personnalisés
- **utils/** : fonctions utilitaires
- **assets/** : images, icônes et autres ressources statiques
